# Logs

Daily learning session logs. One file per session, named `YYYY-MM-DD.md`.

Generated by a custom Claude `daily-learning-log` skill that reads the conversation thread and summarizes it into a structured entry. Posted directly to this repo via Post2myGitHub.

## Fields

| Field | Purpose |
|---|---|
| Topic | What was studied or worked on |
| Core concept | The key insight, in Vincent's own words |
| Connection to main goal | How it serves the AI system, verticals, or job search |
| AI concepts | Technical terms genuinely engaged with (stringent) |
| Career domains | Professional domains honestly touched (stringent) |
| Next action | One concrete next step |
| Sidequest flag | Focus drift check with SERVE/DISTRACT verdict |
| Feynman gap | What needs deeper testing next session |
