# Personal L&D

A public learning and development repository tracking Vincent Kovar's AI skill-building, professional development, and career growth.

This repo is a working record, not a polished portfolio. Entries are generated from real learning sessions and posted directly from Claude.

## Structure

**`/logs`** — Daily learning logs. One file per session, named by date. Generated using a custom Claude skill and posted via Post2myGitHub.

**`/resources`** — Links, references, frameworks, and tools worth keeping. Organized by domain.

**`/projects`** — L&D project artifacts: course notes, capstone work, skill builds, and experiments.

## Context

Vincent is a marketing and AI professional with ~25 years in instructional design and adult education, currently building an AI development skill stack and operating in a CMO/Head of Growth capacity across several ventures.

Active learning tracks: Claude Code, AI agent architecture, GitHub workflows, JavaScript, and agentic workflow design.
